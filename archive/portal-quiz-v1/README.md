# アーカイブ: ポータル＆クイズセッション方式 (v1)

## 概要
3つのポータル（さんすう・こくご・えいご）に近づいて5問のクイズに挑戦する方式。
正解でXP獲得→レベルアップ→建物解放という進行。

## 主な仕組み
- `PORTAL_DEFS`: 3ポータルの定義（位置・教科・アイコン）
- `startQuiz()` → `renderQuestion()` → `answerQuiz()` → `finishQuiz()`
- `worldClears`: 各教科のクリア回数で建物解放条件を管理
- `selectAdaptiveQuestions()`: 正誤実績に基づく適応型問題選択

## 置き換えた理由
資源ブロック採掘方式（Minecraft的な1問1アイテム取得）に変更するため。

## 復元方法
このフォルダの各ファイルをプロジェクトルートにコピーすれば元に戻せる。
